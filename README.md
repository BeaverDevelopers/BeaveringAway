# Editing and Compiling
Please use Godot 4.6.1

# Debug Commands:
- Use P to place an obstacle for the water.
- Use R to reset water simulation.
- Use A to add some extra water to the simulation.
- Use 1 to add a item 1
- Use C to clear item
- Use E to Chop Trees
