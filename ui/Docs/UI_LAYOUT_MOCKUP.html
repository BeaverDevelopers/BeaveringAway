每个 UI 的逻辑

ObjectiveTracker

左上角。默认收纳，只显示当前追踪任务、简短描述和进度。点 + 展开任务列表，每个任务有 Track 按钮。

WorldStatusPanel

右上角。只保留两个进度条：土地恢复度和动物朋友回归度。水 tile 数、长文字焦点不常驻显示。

HealthBar

底部居中，背包上方。做成类似 Minecraft 的心形生命条，之后可以用贴图替换文本心形。

InventoryBar

底部居中。只显示快捷物品/当前资源，不承载完整合成逻辑。

InteractionPrompt

显示在玩家操控角色头顶，跟随玩家与相机移动。接近树、物品、动物时显示 E 提示。

InventoryWindow / Crafting

已经存在。按 Tab 打开完整背包和合成台，所以新 HUD 不再规划单独 CraftingScreen。

ToastFeed

顶部中间。短消息：狐狸偷东西、新配方、新动物回归。几秒后消失。

DialogueBox

底部大框。动物 one-liner 和配方提示时出现，平时隐藏。
